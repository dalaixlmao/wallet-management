## About the Digital Banking Platform

The Digital Banking Platform is a Next.js-based application that provides a modern and secure digital banking experience. The platform offers features such as:

- User authentication and account management
- Balance tracking and monitoring
- Peer-to-peer money transfers
- On-ramp transactions (adding money to your account)
- Transaction history and reporting
- Dashboard with financial analytics

## Technical Stack

The platform is built with the following technologies:

- **Frontend**: Next.js, React, TypeScript, TailwindCSS
- **Backend**: Next.js API Routes, Prisma ORM
- **Authentication**: NextAuth.js
- **Database**: PostgreSQL
- **Visualization**: Chart.js, React-ChartJS-2
