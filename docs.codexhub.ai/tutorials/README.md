# Tutorials

Welcome to the Tutorials section of the Digital Banking Platform documentation. These step-by-step guides will help you learn how to perform common tasks and implement key features.

## Available Tutorials

- [Getting Started with the Platform](./getting-started.md): Set up and run the platform locally
- [Creating Your First Account](./creating-account.md): Step-by-step guide for creating a user account
- [Making Your First Transfer](./making-transfers.md): Learn how to send money to another user
- [Adding Funds to Your Account](./adding-funds.md): Guide for adding money to your account

## How to Use These Tutorials

Each tutorial is designed to be followed sequentially, with clear step-by-step instructions. Code examples are provided where appropriate, and screenshots may be included to illustrate the process.

If you encounter any issues while following these tutorials, please refer to the [Troubleshooting](../guides/user-guide.md#troubleshooting) section of the user guide.
